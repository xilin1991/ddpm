## Prerequisites

The training and testing experiments are conducted using PyTorch 1.8.1 with a single NVIDIA TITAN RTX GPU with 24GB Memory.

- python 3.8
- pytorch 1.8.2
- torchvision 0.9.2
- sklearn 1.2.0
- numpy 1.23.4
- loguru (optional)

## Train

To train the model on a GPUs, you can use:
```bash
bash scripts/train.sh
```
